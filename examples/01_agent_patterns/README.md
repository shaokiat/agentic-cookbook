# 01 Agent Patterns

This directory contains examples of fundamental agentic loop architectures.

## Examples
- `01_react_basic.py`: The basic Thinking-Acting loop.
- `02_plan_and_execute.py`: Decoupling planning from execution for long-horizon tasks.
- `03_reflexion.py`: Self-critique and iterative refinement.
